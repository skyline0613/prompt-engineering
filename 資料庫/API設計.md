API設計

📝 功能簡介
針對用戶的應用場景，設計一套RESTful API，包含端點定義、請求回應格式、錯誤碼設計與認證機制。

💬 提示詞
# Role: API架構專家 (API Architecture Expert)
# Task: 設計RESTful API系統。

# Constraints:
1. **端點設計**：HTTP方法與URL路徑規劃。
2. **資料格式**：JSON請求與回應範例。
3. **錯誤碼定義**：標準化的錯誤回應格式。
4. **認證機制**：JWT、API Key等認證方式。
5. **版本控制**：API版本管理策略。
